# Deskripsi Tugas
Tugas ini bertujuan untuk melakukan **refaktor ringan** pada kode agar lebih stabil dan **mengubah desain/tema website** menjadi dominan pink dengan estetika bernuansa "buku" profesional. 

Mohon kerjakan checklist di bawah ini secara perlahan, satu per satu. Fokus pada kestabilan fungsi sebelum merubah visual.

## 1. Perbaikan Kode (Refactoring)
- [ ] **Paginasi di halaman daftar buku**: Buka file `pages/books/index.vue`. Saat memanggil `useAsyncData('all-books')` lalu `getBooks()`, tolong sesuaikan parameter fungsi `getBooks` agar query menggunakan *limit* atau *pagination* (jangan load ratusan data sekaligus).
- [ ] **Filter Genre**: Saat ini perhitungan genre di `pages/books/index.vue` `allGenres` berjalan manual (melakukan *looping* semua buku). Coba tingkatkan agar tidak membebani komputasi client-side, terutama saat daftar bukunya mulai sangat panjang. (Opsional: ambil nilai genre dari database atau pisahkan logikanya).
- [ ] **Handling Error (API)**: Di dalam `pages/books/[slug].vue`, lihat fungsi pemanggilan `getComments`, `getBooks`, dll. Pastikan ada penangkapan *error* (dengan `try...catch`). Jika gagal, tangani dengan *graceful* (misal: set array menjadi `[]` tanpa error di console).
- [ ] **SEO Meta Tags yang Dinamis**: Di file `pages/books/[slug].vue`, tag `useSeoMeta` saat ini hanya menampilkan title berupa text dari parameter URL `slug`. Tolong edit `useSeoMeta` agar dipanggil secara dinamis menggunakan nilai property `book.value.title` ketika data buku sudah berhasil diambil, sehingga title halamannya berupa judul buku asli.

## 2. Redesign UI: Tema Pink & Estetika Buku
Tolong ubah warna dan tema secara konsisten:
- [ ] **Gunakan Font Buku (Serif)**:
  - Buka file `tailwind.config.ts`. Di dalam `theme.extend.fontFamily`, tambahkan font serif untuk judul/heading seperti `"Playfair Display"` atau `"Lora"`. Pertahankan `"Inter"` untuk teks paragraf dasar (`sans`).
  - Tolong letakkan tag `<style>` atau gunakan `<Link>` API di konfigurasi `nuxt.config.ts` untuk meload font tersebut dari Google Fonts.
- [ ] **Ubah Hero Section**: 
  - Buka file `pages/index.vue`. Ubah warna *gradient background* di bagian atas (Hero area). Ganti kelas seperti `from-gray-900 to-gray-800` menjadi warna elegan turunan warna *Rose* atau *Burgundy* (misal `from-rose-900 to-rose-800`).
  - Berikan kelas font serif (misal `font-serif`) pada heading `<h1>` dan `<h2>` agar terkesan antik/seperti cover buku lama.
- [ ] **Gunakan Warna Dominan Pink untuk Aksen UI**:
  - Di seluruh project, jika ada tombol biru (seperti warna `bg-blue-600`), ubah menjadi turunan warna *Rose* atau *Pink* profesional (misal `bg-rose-600 hover:bg-rose-700`).
  - Ganti style box, detail background, atau progress metrics target baca di `pages/index.vue` untuk mengikuti nuansa *pink/blush*.
  - Buka `components/ui/StarRating.vue`. Ubah bagian pewarnaan bintang aktif; alih-alih warna standard emas/kuning, cobalah gunakan warna bernuansa elegan yang cocok dengan tema *rose* (misal `text-rose-400`).
  - Buka `components/ui/GenreBadge.vue`. Beri warna kaleng yang pastel namun jelas terbaca, misalnya dengan `bg-rose-50 text-rose-800 border border-rose-200`.

## Aturan Pengerjaan:
1. Pastikan Anda sudah menjalankan environment dengan `npm run dev`.
2. Jangan merusak koneksi Supabase. Selalu cek bahwa data daftar buku masih dapat dimuat.
3. Kerjakan dan pastikan aplikasinya tidak error setiap menyelesaikan satu poin checklist.
