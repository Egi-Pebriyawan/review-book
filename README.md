# review-book

Scaffold Nuxt 3 + Tailwind + Supabase project files as requested in Issue #1.

Quick start:

```powershell
npm install
npm run dev
```

Files added:

- package.json
- nuxt.config.ts
- tailwind.config.ts
- postcss.config.cjs
- app.vue
- pages/index.vue
- assets/css/tailwind.css

After `npm install`, `npm run dev` should start the dev server at http://localhost:3000
